Project Name: Cloud-Based Database Deployment Using Amazon RDS

Project Scope: This project involves deploying and managing a cloud-based relational database using Amazon RDS (Relational Database Service). The implementation includes creating a custom VPC, configuring public and private subnets, setting up route tables and an internet gateway, launching an Amazon Linux EC2 instance, creating a MySQL database instance on Amazon RDS, connecting to the database using the MariaDB client, and performing Create, Read, Update, and Delete (CRUD) operations. The project demonstrates secure database deployment, connectivity, and management within the AWS cloud environment.
